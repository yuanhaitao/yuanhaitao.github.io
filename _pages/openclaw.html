🤔 OpenClaw 到底是啥？一句话解释

OpenClaw = 住在你电脑里的 AI 管家。你用手机给它发消息，它就帮你在电脑上干活。

打个比方：

  • ChatGPT 就像一个很聪明但没有手的朋友 — 你问它什么它都能回答，但它不能帮你做任何事，只能"动嘴"
  • OpenClaw 就像一个又聪明又有手的机器人管家 — 不仅能回答问题，还能直接在你电脑上操作：整理文件、写文档、发邮件、做表格、甚至帮你写代码

具体能帮你做什么？

  • 帮你处理文件 — "帮我把桌面上的文件按类型分类整理"
  • 帮你写东西 — "帮我写一篇 800 字的读书笔记"
  • 帮你上网查资料 — "帮我搜一下明天的天气预报"
  • 帮你做定时任务 — "每天早上 7 点提醒我背单词"
  • 记住你的一切 — 越用越懂你，不会忘记你之前说过的话

而且它是完全免费、开源的（GitHub 上 24 万星），数据都存在你自己电脑里，不用担心隐私问题。

命令行入门（完全小白必看）

安装 OpenClaw 需要用一小会儿命令行（也叫终端 / Terminal）。这可能是你第一次用，但真的非常简单。你只需要用它安装一次，以后日常使用都是在聊天软件/网页里操作的。

先深呼吸放松一下！命令行看起来黑乎乎的很吓人，但它其实特别"笨" — 你不输入东西它什么都不会做。就算你乱打一通也不会弄坏电脑。而且你只需要用它复制粘贴几条命令就行，根本不需要记住任何东西。

🤔 命令行是什么？30 秒看懂

你平时怎么操作电脑？用鼠标点来点去对吧。命令行就是另一种操作电脑的方式 — 用打字代替鼠标点击。

举个例子，同样是"打开桌面上的一个文件夹"：

操作鼠标党的做法命令行的做法
打开文件夹双击文件夹图标输入 cd 文件夹名
看看里面有啥打开文件夹看一眼输入 ls
新建文件夹右键 → 新建文件夹输入 mkdir 文件夹名

是不是没那么可怕？而且你只需要会复制粘贴就行，不需要手打。

📖 第一步：打开命令行（跟着做，10 秒搞定）

最快方式（3 秒）：

  1. 同时按下键盘上的 Command（⌘）键和空格键
  2. 屏幕中间会弹出一个搜索框
  3. 输入 Terminal（或者输"终端"也行）
  4. 按回车 → 一个黑/白色的窗口弹出来了，这就是命令行！

也可以打开 访达(Finder) → 应用程序 → 实用工具 → 双击 终端(Terminal)。

推荐方式：安装 Git Bash（OpenClaw 在 Windows 上推荐用这个）

  1. 去 git-scm.com/downloads 下载安装 Git for Windows
  2. 安装时一路点"Next"用默认设置就行
  3. 安装完后，在桌面右键 → 你会看到"Git Bash Here" → 点它就打开了

备选方式：在键盘上按 Win 键，输入 PowerShell，点击打开。

同时按下 Ctrl + Alt + T 就打开了。或者在应用菜单中搜索 "Terminal"。

⌨️ 你只需要记住的几个命令

命令作用举例
cd 目录名 进入某个文件夹 cd ~/Desktop
ls 查看当前文件夹里有什么 ls
pwd 显示当前路径 pwd → /Users/zhangsan

记住这三个就够了。安装好 OpenClaw 之后，日常使用是通过聊天软件，不需要再碰终端。

看到 $ 或 % 符号？ 那是命令行在跟你说"我准备好了，你可以输入了"。你不需要自己打这个符号，只打后面的内容就行。

怕输错？完全不用怕！

  • 输错了 → 按 Ctrl + C 取消，重新来
  • 屏幕乱了 → 输入 clear 清屏
  • 卡住了 → 按 Ctrl + C 强制停止
  • 想退出 → 输入 exit 或直接关掉窗口

命令行不会因为你输错而弄坏电脑。最坏的情况就是报错，然后你重新输就好了。放心大胆地试！

好消息：你只需要用命令行安装这一次。装好之后，日常使用 OpenClaw 都是在网页/聊天软件里操作的，再也不用碰命令行了（除非你想）。

安装 OpenClaw

给纯小白的最短路径（3 步搞定）：

  1. 打开终端（上面教过了）
  2. 复制下面"一键脚本"里的那一条命令，粘贴到终端里，按回车
  3. 等 2-5 分钟，跟着弹出来的向导选就行

就这么简单。下面是详细说明，遇到问题再看。

最简单的方式！一条命令搞定，会自动检测系统、安装 Node.js 和所有依赖。

打开终端，运行安装命令

bash
curl -fsSL https://openclaw.ai/install.sh | bash

PowerShell（以管理员身份运行）：

powershell
irm https://openclaw.ai/install.ps1 | iex

等待安装完成

脚本会自动安装 Node.js（如果没有的话）和 OpenClaw，通常需要 2-5 分钟。安装完成后会自动进入配置向导。

Terminal
$ curl -fsSL https://openclaw.ai/install.sh | bash Detecting system... macOS arm64 Installing Node.js 22... ✅ Installing OpenClaw v2026.3.2... ✅ 🦞 OpenClaw installed successfully! Starting onboard wizard...

如果你已经有 Node.js 22+，可以直接用 npm 安装。

先确认 Node.js 版本

bash
node -v # 必须是 v22.x.x 或更高

如果版本低于 22，需要先升级 Node.js：

bash — Mac (Homebrew)
brew install node@22

安装 OpenClaw

bash
npm install -g openclaw@latest

国内用户网速慢？加上淘宝镜像源：

bash
npm install -g openclaw@latest --registry=https://registry.npmmirror.com

启动配置向导

bash
openclaw onboard --install-daemon

这条命令做了什么？

  • openclaw onboard — 启动交互式配置向导
  • --install-daemon — 同时安装后台服务，让 OpenClaw 开机自动运行

适合有服务器（VPS）的用户，或者想在 Docker 容器中隔离运行的用户。需要先安装 Docker。

克隆仓库并运行安装脚本

bash
git clone https://github.com/openclaw/openclaw.git cd openclaw ./docker-setup.sh

或使用预构建镜像

bash
export OPENCLAW_IMAGE="ghcr.io/openclaw/openclaw:latest" ./docker-setup.sh

国内用户如果拉取 GitHub 镜像慢，可以使用汉化版 Docker 镜像：

bash — 汉化版
docker pull 1186258278/openclaw-zh:latest

验证安装成功

bash
openclaw --version
Terminal
$ openclaw --version v2026.3.2 ← 看到版本号就说明安装成功了！

安装遇到问题？

  • EACCES: permission denied — 在命令前加 sudo，或配置 npm 权限
  • command not found — 关闭终端重新打开，或检查 npm 全局路径是否在 PATH 中
  • sharp 安装失败 — 运行 SHARP_IGNORE_GLOBAL_LIBVIPS=1 npm install -g openclaw@latest
  • 有其他问题 — 运行 openclaw doctor --fix 自动诊断修复

配置向导（最重要！）

安装完成后会自动进入 openclaw onboard 配置向导。这是一个交互式的问答流程 — 它问你问题，你选答案，就像填问卷一样简单。跟着走就行，选错了也没关系，以后随时可以重新配。

小白最快通关路线：风险提示选 Yes → 安装模式选 QuickStart → AI 模型选 Qwen（不需要 API Key）→ 聊天平台选 None → Skills 跳过 → 完成！全程 1 分钟。

接受风险提示

OpenClaw 会提醒你它有权限操作你的电脑文件。阅读后选择 Yes 继续。

选择安装模式

新手直接选 QuickStart（快速启动），它会用推荐的默认配置。

选择 AI 模型

这一步选择给 OpenClaw "装哪个大脑"。国内推荐：

模型特点价格
DeepSeek 国内直连，性价比最高 极便宜（几毛钱/天）
通义千问 Qwen 阿里出品，中文理解好，有免费额度 有免费额度
Kimi (Moonshot) 长文本处理专家 中等
Claude / GPT 能力最强，但需要翻墙或代理 较贵

新手推荐：选 Qwen（通义千问）最省心。选了之后会跳转到登录页面，授权一下就行，不需要自己去申请 API Key。选 DeepSeek 最便宜但需要自己去 DeepSeek 平台 注册获取 API Key。

输入 API Key

根据你选的模型，输入对应的 API Key。向导会提示你去哪里获取。

选择聊天平台

选择你想通过哪个聊天软件和 OpenClaw 对话（国内用户推荐）：

  • 飞书（Lark） — 国内推荐，Bot 设置简单
  • 钉钉（DingTalk） — 大部分人都有，企业常用
  • None — 先跳过也行！后面可以直接用浏览器打开 Web UI 使用，完全不需要配聊天软件

小白建议：如果你不想折腾聊天平台配置，直接选 None。安装完成后用浏览器访问 http://localhost:18789 就能用了，和网页版 ChatGPT 一样方便。等熟悉了再去配飞书/钉钉也不迟。

配置技能（Skills）

用空格键勾选你需要的技能。不确定的话直接跳过，后面随时可以加。

完成！

向导结束后，OpenClaw 会自动启动 Gateway 服务在后台运行。你现在可以去聊天软件里给它发消息了！

接入聊天平台

OpenClaw 支持多种聊天平台。国内用户不需要翻墙，下面的方案都可以直接用。

最简单的方式：先用 Web UI！OpenClaw 安装完成后自带一个网页界面，在浏览器打开 http://localhost:18789 就能直接对话，不需要配任何聊天平台。等你用熟了，再考虑接入飞书/钉钉，这样出门在外也能用手机控制 AI。

Web UI 是 OpenClaw 自带的网页界面，不需要任何额外配置，安装完就能用。

打开浏览器

在浏览器地址栏输入：

浏览器地址栏
http://localhost:18789

输入 Token 认证

首次访问需要输入 Token。这个 Token 在你安装 OpenClaw 完成时会在终端里显示，类似这样：

终端输出示例
Web UI: http://localhost:18789?token=abc123def456

直接点击那个带 ?token=xxx 的完整链接，或者手动复制 token 粘贴到页面上。

开始对话

配置 AI 模型（重点！）

OpenClaw 本身只是一个"身体"，它需要一个 AI 大脑才能工作。你需要给它接一个"大脑"，也就是 AI 大模型。

🤔 先搞懂：什么是 API Key？

你可以把 API Key 理解为一把"钥匙"。AI 大模型（比如 DeepSeek、通义千问）运行在云端服务器上，你需要一把钥匙才能用它。这把钥匙就是 API Key —— 一串像密码一样的字符串。

流程是这样的：

你发消息
→
OpenClaw
带着你的 API Key
→
AI 模型服务器
验证钥匙，返回回答

所以你需要做两件事：1) 去 AI 平台注册获取 API Key；2) 把 API Key 填到 OpenClaw 的配置里。下面手把手教你。

适合谁？先想免费试试、不想花钱、或者对隐私要求极高的用户。以下所有方案国内都能直接用，不需要翻墙。

选项 1：通义千问 Qwen — 新手首选，注册就送免费额度

阿里云的通义千问（Qwen）提供了免费的 API 调用额度。注册就送，个人日常使用完全够用。不需要装任何东西，云端运行，效果很好，国内直连，速度快。

免费额度有多少？新用户注册通常会送 100 万～数百万 Token 的免费额度（具体以平台最新活动为准）。日常聊天使用的话，够你免费用好几个月。

打开阿里云百炼平台

在浏览器中打开：bailian.console.aliyun.com

如果你有支付宝或淘宝账号，可以直接扫码登录，不需要额外注册。没有的话用手机号注册一个阿里云账号也很快。

开通百炼服务

首次登录会提示你"开通百炼大模型服务"，点同意并开通就行（免费开通，不会扣钱）。

创建 API Key

开通后，在页面左上角或左侧菜单找到 "API-KEY 管理"（也可能叫"API Key"），点进去：

  • 点 "创建新的 API-KEY"
  • 系统会生成一串以 sk- 开头的字符串，这就是你的 API Key
  • 重要：复制保存好！这个 Key 只会显示一次，关掉页面就看不到了。建议先粘贴到记事本里

API Key 是你的"密码"，不要泄露给别人！谁拿到这个 Key 就能用你的额度调用 AI。不要发到群里、不要截图发朋友圈。

把 API Key 配置到 OpenClaw

回到终端，输入下面的命令。注意把 sk-你的API Key 替换成你刚才复制的那串真实的 Key：

bash
openclaw config set 'models.providers.qwen' --json '{ "baseUrl": "https://dashscope.aliyuncs.com/compatible-mode/v1", "apiKey": "sk-你的API Key", "models": { "default": "qwen-plus" } }'

逐行解释这个命令：

  • baseUrl — AI 模型服务器的地址，这里是阿里云的地址，不要改
  • apiKey — 你刚才在阿里云创建的那串 Key，替换成你自己的
  • default — 使用哪个模型，qwen-plus 是性价比最高的选择

重启 OpenClaw 并测试

bash
openclaw gateway restart

然后去 Web UI（http://localhost:18789）或者聊天软件里发一条消息，收到回复就说明配置成功了！

选项 3：Ollama — 在自己电脑上跑 AI（完全离线）

Ollama 让你在自己电脑上运行开源大模型，完全免费、完全离线、数据不出本地。适合对隐私要求极高、或者网络条件不好的用户。

硬件要求比较高！本地跑 AI 需要较好的电脑配置：

  • 8GB 内存 — 勉强能跑 7B 小模型，效果一般，回复慢
  • 16GB 内存 — 可以跑 7B-14B 模型，效果还行
  • 32GB+ 内存 — 可以跑 32B 模型，效果很好
  • 有 Apple M 系列芯片（MacBook Air M1/M2/M3/M4）或 NVIDIA 显卡 会快很多

如果你的电脑配置不高，建议用上面的通义千问或硅基流动方案（云端运行，对电脑没要求）。

安装 Ollama

安装完成后，在启动台（Launchpad）里打开 Ollama，它会在菜单栏出现一个小图标，说明已经在后台运行了。

bash
curl -fsSL https://ollama.com/install.sh | sh

去 ollama.com/download 下载 Windows 版安装包（.exe 文件），双击运行安装。

下载一个 AI 模型

打开终端，运行以下命令下载模型（第一次需要下载几个 GB 的文件，耐心等待）：

bash — 选一个运行即可

下载完成后可以先试试效果：输入 ollama run qwen2.5:7b，然后直接打字对话。输入 /bye 退出。

配置 OpenClaw 使用 Ollama

Ollama 安装后会在你电脑本地启动一个 API 服务（地址是 http://localhost:11434），OpenClaw 可以直接连上去用：

bash
openclaw config set 'models.providers.ollama' --json '{ "baseUrl": "http://localhost:11434/v1", "apiKey": "ollama", "models": { "default": "qwen2.5:7b" } }'

apiKey 这里随便填什么都行（本地运行不需要密钥），但不能留空，所以写个 ollama 占位。

重启并测试

bash
# 确认 Ollama 在运行（应该能看到你下载的模型） ollama list # 重启 OpenClaw openclaw gateway restart

去 Web UI 发条消息试试，能收到回复就说明成功了！本地模型回复速度取决于你的电脑配置，可能需要等几秒到几十秒。

配置出问题了？排查清单：

项目最低要求推荐配置

安全注意事项

OpenClaw 非常强大，但能力越大，风险越大。请注意以下安全事项：

⚠️ 必须知道的风险

  • 文件操作风险 — OpenClaw 可以读写你电脑上的文件，操作不当可能导致数据丢失
  • 第三方 Skill 风险 — 社区提交的 Skill 可能包含恶意代码，Cisco 安全团队已发现过数据泄露的案例
  • 上下文压缩风险 — 对话过长时 AI 可能"忘记"安全规则，曾有用户报告 OpenClaw 自动删除了邮件
  • API Key 泄露风险 — 你的配置文件中包含 API Key 等敏感信息

🛡️ 安全最佳实践

  • 使用 Docker 沙盒 — 在容器中运行 OpenClaw，限制它对主机的访问
  • 只安装可信的 Skill — 优先使用官方推荐或高星标的 Skill
  • 定期检查操作日志 — 看看 OpenClaw 做了什么
  • 重要数据先备份 — 在让 OpenClaw 操作重要文件之前，先做好备份
  • 不要给太高权限 — 避免用 root / 管理员权限运行 OpenClaw
  • 设置操作审批 — 让 OpenClaw 在执行危险操作前先征求你的同意

常见问题

Q：我完全不会编程，能用 OpenClaw 吗？ ▼

完全可以！OpenClaw 的日常使用就是发消息聊天，和用微信一样简单。你说"帮我整理桌面文件"，它就去帮你整理。不需要写任何代码。唯一需要用到命令行的就是安装那一次，跟着教程复制粘贴就行。

Q：我是初中生/高中生，可以用来干嘛？ ▼

太多了！比如：

  • 学习助手：帮你整理笔记、出练习题、解释难题、翻译英语
  • 写作帮手：帮你改作文、写读书笔记、做 PPT 大纲
  • 学编程：让它教你写 Python 小游戏，边玩边学
  • 做小项目：帮你做个人网页、小工具、数据分析
  • 定时提醒：每天提醒你背单词、做作业

而且这些全都免费。用通义千问的免费额度就够日常学习用好几个月。

Q：OpenClaw 和 Clawdbot / Moltbot 是什么关系？ ▼

它们是同一个项目的不同名字。开发者 Peter Steinberger 最初把项目命名为 Clawdbot（龙虾谐音 Claude），后来因为 Anthropic 的商标投诉，先改名为 Moltbot（龙虾脱壳之意），最终定名为 OpenClaw。所以你看到网上提到这三个名字，说的都是同一个东西。

Q：OpenClaw 是免费的吗？ ▼

OpenClaw 本身是完全免费开源的。但你需要自己付 AI 模型的 API 调用费用。比如用 DeepSeek 的话，大概几毛钱一天；用 GPT-4 或 Claude 的话就贵一些。选择通义千问 Qwen 还有免费额度可以用。

Q：关掉电脑后 OpenClaw 还能用吗？ ▼

如果 OpenClaw 安装在你的个人电脑上，关机后就不能用了。如果你想 24 小时运行，有两种方案：

  • 买一台便宜的 Mac Mini 或树莓派，专门跑 OpenClaw
  • 租一台云服务器（VPS），用 Docker 部署
Q：OpenClaw 安全吗？我的数据会不会泄露？ ▼

OpenClaw 本身运行在你自己的设备上，数据不会传到 OpenClaw 的服务器（因为它没有中心服务器）。但有两个注意点：

  • 你和 AI 的对话内容会发送到 AI 模型提供商（如 DeepSeek、OpenAI），这和你直接用 ChatGPT 是一样的
  • 第三方 Skill 可能有安全风险，请只安装可信来源的 Skill

建议用 Docker 沙盒运行，并定期检查操作日志。

Q：onboard 向导出错了怎么办？ ▼

可以重新运行向导：

bash
openclaw onboard

或者运行诊断工具：

bash
openclaw doctor --fix
Q：Moltbook 是什么？ ▼

Moltbook 是一个 OpenClaw Agent 自己创建的"AI 社交网络"。上面全是 AI 机器人在发帖、评论、互相点赞，人类只能围观不能参与。它一度病毒式传播，累计注册了 160 万个 AI Bot，产生了 750 万条 AI 生成的帖子。这个现象本身就展示了 OpenClaw 的强大能力。

Q：如何更新 OpenClaw？ ▼
bash
openclaw update

或者用 npm 重新安装最新版：

bash
npm install -g openclaw@latest

OpenClaw 小白教程

从零开始，手把手教你安装、配置和使用 OpenClaw。在微信/Telegram/WhatsApp 里拥有一个真正能帮你干活的 AI 私人助手。

🤔 OpenClaw 到底是啥？

简单说，OpenClaw 就是一个运行在你自己电脑上的 AI 私人助手。它最大的特点是：你在聊天软件里给它发消息，它就能帮你操作电脑。

  • 帮你处理文件 — 批量重命名、整理、转格式
  • 帮你写东西 — 写报告、邮件、代码
  • 帮你跑命令 — 在你电脑上执行各种操作
  • 帮你上网查资料 — 浏览器自动化搜索和采集
  • 帮你做定时任务 — 定期执行某些操作（类似定闹钟）
  • 记住你的一切 — 长期记忆，越用越懂你

它和 ChatGPT 最大的不同：ChatGPT 只会"说"，OpenClaw 会"做"。你让它"帮我把邮箱里的发票都整理到一个文件夹"，它真的会帮你做，而不只是告诉你怎么做。

和 Claude Code 有什么区别？

  • Claude Code：在终端命令行里直接对话，主要面向编程场景
  • OpenClaw：通过聊天软件（Telegram、WhatsApp 等）对话，面向所有日常场景，是一个全能 AI 管家

如果你主要做编程 → 用 Claude Code；如果你想要一个日常生活全能助手 → 用 OpenClaw。当然两个都用也行！

命令行入门（完全小白必看）

OpenClaw 的安装和配置需要用到命令行（也叫终端 / Terminal）。如果你从来没用过命令行，别慌，看完这一节就够了。

🤔 命令行是什么？

命令行就是一个用打字代替鼠标点击来操作电脑的工具。你在里面输入一条"命令"，按下回车，电脑就会执行。

比如：

  • 你想打开某个文件夹？输 cd 文件夹名（相当于双击文件夹）
  • 你想看文件夹里有什么？输 ls（相当于打开文件夹看一眼）
  • 你想创建一个新文件夹？输 mkdir 文件夹名（相当于右键→新建文件夹）

别怕，你只需要会打开终端和输入几个简单命令就行。安装完 OpenClaw 之后，日常使用都是在聊天软件里操作的。

📖 怎么打开命令行？

按 Command + 空格，输入 Terminal（或"终端"），按回车即可打开。

也可以打开 访达 → 应用程序 → 实用工具 → 双击 终端。

推荐安装 Git for Windows，然后使用 Git Bash。

或者按 Win + X，选择 "Windows Terminal" 或 "PowerShell"。

按 Ctrl + Alt + T 打开终端。或在应用菜单中搜索 "Terminal"。

⌨️ 你只需要记住的几个命令

命令作用举例
cd 目录名 进入某个文件夹 cd ~/Desktop
ls 查看当前文件夹里有什么 ls
pwd 显示当前路径 pwd → /Users/zhangsan

记住这三个就够了。安装好 OpenClaw 之后，日常使用是通过聊天软件，不需要再碰终端。

看到 $ 或 % 符号？ 那是命令行的"提示符"，表示"我准备好了"。你不需要输入这个符号，直接输入后面的命令就行。输错了按 Ctrl + C 取消。

准备工作

在安装之前，请确保你有以下东西：

📦 你需要准备的东西

项目说明怎么获得
电脑 macOS / Linux / Windows 都行 你正在用的就行
Node.js 22+ OpenClaw 依赖 Node.js 运行 见下方安装教程
AI 模型的 API Key 让 AI 工作的密钥 DeepSeek / 通义千问 / Claude 等
聊天软件账号 用来和 OpenClaw 对话 Telegram（推荐）/ WhatsApp / 飞书

国内用户推荐方案：

  • AI 模型：选 DeepSeek（性价比最高）或 通义千问 Qwen（免费额度多）
  • 聊天平台：选 Telegram（最简单、国内可用）或 飞书（国内办公常用）

关于 Node.js 版本：OpenClaw 要求 Node.js v22 或以上（比 Claude Code 要求更高）。如果你之前装过旧版 Node.js，需要升级。

2

安装 OpenClaw

最简单的方式！一条命令搞定，会自动检测系统、安装 Node.js 和所有依赖。

打开终端，运行安装命令

bash
curl -fsSL https://openclaw.ai/install.sh | bash

PowerShell（以管理员身份运行）：

powershell
irm https://openclaw.ai/install.ps1 | iex

等待安装完成

脚本会自动安装 Node.js（如果没有的话）和 OpenClaw，通常需要 2-5 分钟。安装完成后会自动进入配置向导。

Terminal
$ curl -fsSL https://openclaw.ai/install.sh | bash Detecting system... macOS arm64 Installing Node.js 22... ✅ Installing OpenClaw v2026.3.2... ✅ 🦞 OpenClaw installed successfully! Starting onboard wizard...

如果你已经有 Node.js 22+，可以直接用 npm 安装。

先确认 Node.js 版本

bash
node -v # 必须是 v22.x.x 或更高

如果版本低于 22，需要先升级 Node.js：

bash — Mac (Homebrew)
brew install node@22

安装 OpenClaw

bash
npm install -g openclaw@latest

国内用户网速慢？加上淘宝镜像源：

bash
npm install -g openclaw@latest --registry=https://registry.npmmirror.com

启动配置向导

bash
openclaw onboard --install-daemon

这条命令做了什么？

  • openclaw onboard — 启动交互式配置向导
  • --install-daemon — 同时安装后台服务，让 OpenClaw 开机自动运行

适合有服务器（VPS）的用户，或者想在 Docker 容器中隔离运行的用户。需要先安装 Docker。

克隆仓库并运行安装脚本

bash
git clone https://github.com/openclaw/openclaw.git cd openclaw ./docker-setup.sh

或使用预构建镜像

bash
export OPENCLAW_IMAGE="ghcr.io/openclaw/openclaw:latest" ./docker-setup.sh

国内用户如果拉取 GitHub 镜像慢，可以使用汉化版 Docker 镜像：

bash — 汉化版
docker pull 1186258278/openclaw-zh:latest

验证安装成功

bash
openclaw --version
Terminal
$ openclaw --version v2026.3.2 ← 看到版本号就说明安装成功了！

安装遇到问题？

  • EACCES: permission denied — 在命令前加 sudo，或配置 npm 权限
  • command not found — 关闭终端重新打开，或检查 npm 全局路径是否在 PATH 中
  • sharp 安装失败 — 运行 SHARP_IGNORE_GLOBAL_LIBVIPS=1 npm install -g openclaw@latest
  • 有其他问题 — 运行 openclaw doctor --fix 自动诊断修复
3

配置向导（最重要！）

安装完成后会自动进入 openclaw onboard 配置向导。这一步决定了 OpenClaw 用哪个 AI 大脑、通过哪个聊天平台和你沟通。跟着向导一步一步来就行。

接受风险提示

OpenClaw 会提醒你它有权限操作你的电脑文件。阅读后选择 Yes 继续。

选择安装模式

新手直接选 QuickStart（快速启动），它会用推荐的默认配置。

选择 AI 模型

这一步选择给 OpenClaw "装哪个大脑"。国内推荐：

模型特点价格
DeepSeek 国内直连，性价比最高 极便宜（几毛钱/天）
通义千问 Qwen 阿里出品，中文理解好，有免费额度 有免费额度
Kimi (Moonshot) 长文本处理专家 中等
Claude / GPT 能力最强，但需要翻墙或代理 较贵

新手推荐：选 Qwen（通义千问）最省心。选了之后会跳转到登录页面，授权一下就行，不需要自己去申请 API Key。选 DeepSeek 最便宜但需要自己去 DeepSeek 平台 注册获取 API Key。

输入 API Key

根据你选的模型，输入对应的 API Key。向导会提示你去哪里获取。

选择聊天平台

选择你想通过哪个聊天软件和 OpenClaw 对话：

  • Telegram — 最推荐，设置最简单
  • WhatsApp — 扫码绑定即可
  • 飞书 — 国内办公常用
  • None — 先跳过，之后再配（可以通过 Web UI 使用）

配置技能（Skills）

用空格键勾选你需要的技能。不确定的话直接跳过，后面随时可以加。

完成！

向导结束后，OpenClaw 会自动启动 Gateway 服务在后台运行。你现在可以去聊天软件里给它发消息了！

接入聊天平台

OpenClaw 支持非常多的聊天平台。下面介绍最常用的几个：

Telegram 是最适合 OpenClaw 的平台，设置最简单。

创建 Telegram Bot

在 Telegram 中搜索 @BotFather，发送 /newbot，按提示操作：

  • 输入 Bot 名称（例如：我的AI助手）
  • 输入 Bot 用户名（必须以 bot 结尾，例如：myai_assistant_bot）
  • BotFather 会给你一个 Bot Token（一串数字和字母），复制保存好

配置到 OpenClaw

如果在 onboard 向导中已经配置过，这步可以跳过。否则运行：

bash
openclaw config set 'channels.telegram.botToken' '你的Bot Token'

然后重启 Gateway：

bash
openclaw gateway restart

开始使用

在 Telegram 中搜索你刚创建的 Bot，发送 /start，然后就可以正常对话了！

WhatsApp 使用 WhatsApp Web 协议（Baileys），需要扫码绑定。

在 onboard 中选择 WhatsApp

向导会显示一个 QR 码，用你手机的 WhatsApp 扫描即可绑定。

开始使用

绑定后，你在 WhatsApp 中给 OpenClaw 发消息就能收到回复了。

官方建议 WhatsApp 使用单独的手机号，避免影响你的日常 WhatsApp 使用。

飞书（Lark）适合国内办公用户。需要在飞书开放平台创建自建应用。

创建飞书应用

登录 飞书开放平台，创建一个自建应用，获取 App ID 和 App Secret。

配置到 OpenClaw

在 onboard 向导中选择飞书，按提示输入 App ID、App Secret 等信息。

OpenClaw 还支持这些平台：Discord、Slack、Signal、iMessage、Google Chat、Microsoft Teams、Matrix、IRC、LINE、Mattermost 等 20+ 个平台。详情见 官方文档。

技能（Skills）

Skills 是 OpenClaw 的"超能力模块"。每个 Skill 让 OpenClaw 多一项本领。

🧩 什么是 Skill？

Skill 就像手机里的 App。OpenClaw 本身是操作系统，Skill 是安装在上面的应用程序。

  • 邮件 Skill — 让 OpenClaw 能收发和管理邮件
  • 日历 Skill — 让 OpenClaw 能查看和管理你的日程
  • 浏览器 Skill — 让 OpenClaw 能自动浏览网页
  • 文件管理 Skill — 让 OpenClaw 能操作你的文件
  • 代码 Skill — 让 OpenClaw 能写代码和执行脚本

📦 怎么安装 Skill？

你可以在 skills.sh 上搜索你想要的技能，或者直接让 OpenClaw 帮你安装：

Telegram — OpenClaw
你： 帮我安装一个天气查询的技能 🦞 OpenClaw： 已安装 weather-skill ✅ 现在你可以问我天气了！试试发"今天北京天气怎么样"

也可以用命令行管理：

bash
# 查看已安装的技能 openclaw skills # 安装新技能 openclaw skills install weather-skill

安全提醒：只从可信来源安装 Skill。第三方 Skill 有可能包含恶意代码。安装前建议查看 Skill 的源代码和社区评价。

配置 AI 模型

OpenClaw 支持接入多种 AI 模型。你可以随时切换"大脑"。

获取 DeepSeek API Key

去 platform.deepseek.com 注册账号，在"API Keys"页面创建一个 Key。需要先充值（最低 10 元就够用很久）。

配置到 OpenClaw

bash
openclaw config set 'models.providers.deepseek' --json '{ "baseUrl": "https://api.deepseek.com", "apiKey": "你的DeepSeek API Key", "models": { "default": "deepseek-chat", "reasoning": "deepseek-reasoner" } }'

注意：模型 ID 必须使用 deepseek-chat，不要自己改名字，否则会调用失败。账户余额必须大于 0。

最省心的选择！如果在 onboard 向导中选了 Qwen，它会自动跳转到登录页面授权，不需要手动输入 API Key。

如果需要手动配置，去 阿里云百炼平台 获取 API Key，然后在 onboard 中选择 Qwen 并输入。

如果你有 Anthropic 或 OpenAI 的 API Key（可能需要翻墙或使用代理），在 onboard 向导中选择 Claude 或 OpenAI，输入对应的 API Key 即可。

如果你已经在用第三方 API 代理来跑 Claude Code，同样的 API Key 和 Base URL 通常也可以用在 OpenClaw 上。

Docker 部署（进阶）

如果你有一台服务器（VPS/云主机），可以把 OpenClaw 部署到服务器上，这样即使你的电脑关了，AI 助手也在 24 小时运行。

🐳 为什么用 Docker？

  • 环境隔离 — 不会影响服务器上的其他程序
  • 一键部署 — 不需要手动安装各种依赖
  • 安全性更好 — OpenClaw 运行在容器沙盒中

📋 系统要求

项目最低要求推荐配置
内存2 GB4 GB
磁盘10 GB20 GB
DockerDocker Engine + Compose v2最新版

🚀 快速部署步骤

bash — 在服务器上执行
# 1. 克隆仓库 git clone https://github.com/openclaw/openclaw.git cd openclaw # 2. 运行安装脚本（会自动拉镜像、配置、启动） ./docker-setup.sh # 3. 进入容器完成配置向导 docker exec -it openclaw openclaw onboard

配置完成后，OpenClaw 的 Web UI 默认在 http://服务器IP:18789 上运行。

安全提醒：如果服务器有公网 IP，请务必配置防火墙规则，限制 18789 端口的访问。不要把 OpenClaw 的 Web UI 直接暴露在公网上。

安全注意事项

OpenClaw 非常强大，但能力越大，风险越大。请注意以下安全事项：

⚠️ 必须知道的风险

  • 文件操作风险 — OpenClaw 可以读写你电脑上的文件，操作不当可能导致数据丢失
  • 第三方 Skill 风险 — 社区提交的 Skill 可能包含恶意代码，Cisco 安全团队已发现过数据泄露的案例
  • 上下文压缩风险 — 对话过长时 AI 可能"忘记"安全规则，曾有用户报告 OpenClaw 自动删除了邮件
  • API Key 泄露风险 — 你的配置文件中包含 API Key 等敏感信息

🛡️ 安全最佳实践

  • 使用 Docker 沙盒 — 在容器中运行 OpenClaw，限制它对主机的访问
  • 只安装可信的 Skill — 优先使用官方推荐或高星标的 Skill
  • 定期检查操作日志 — 看看 OpenClaw 做了什么
  • 重要数据先备份 — 在让 OpenClaw 操作重要文件之前，先做好备份
  • 不要给太高权限 — 避免用 root / 管理员权限运行 OpenClaw
  • 设置操作审批 — 让 OpenClaw 在执行危险操作前先征求你的同意

常见问题

Q：OpenClaw 和 Clawdbot / Moltbot 是什么关系？ ▼

它们是同一个项目的不同名字。开发者 Peter Steinberger 最初把项目命名为 Clawdbot（龙虾谐音 Claude），后来因为 Anthropic 的商标投诉，先改名为 Moltbot（龙虾脱壳之意），最终定名为 OpenClaw。所以你看到网上提到这三个名字，说的都是同一个东西。

Q：OpenClaw 是免费的吗？ ▼

OpenClaw 本身是完全免费开源的。但你需要自己付 AI 模型的 API 调用费用。比如用 DeepSeek 的话，大概几毛钱一天；用 GPT-4 或 Claude 的话就贵一些。选择通义千问 Qwen 还有免费额度可以用。

Q：关掉电脑后 OpenClaw 还能用吗？ ▼

如果 OpenClaw 安装在你的个人电脑上，关机后就不能用了。如果你想 24 小时运行，有两种方案：

  • 买一台便宜的 Mac Mini 或树莓派，专门跑 OpenClaw
  • 租一台云服务器（VPS），用 Docker 部署
Q：OpenClaw 安全吗？我的数据会不会泄露？ ▼

OpenClaw 本身运行在你自己的设备上，数据不会传到 OpenClaw 的服务器（因为它没有中心服务器）。但有两个注意点：

  • 你和 AI 的对话内容会发送到 AI 模型提供商（如 DeepSeek、OpenAI），这和你直接用 ChatGPT 是一样的
  • 第三方 Skill 可能有安全风险，请只安装可信来源的 Skill

建议用 Docker 沙盒运行，并定期检查操作日志。

Q：onboard 向导出错了怎么办？ ▼

可以重新运行向导：

bash
openclaw onboard

或者运行诊断工具：

bash
openclaw doctor --fix
Q：Moltbook 是什么？ ▼

Moltbook 是一个 OpenClaw Agent 自己创建的"AI 社交网络"。上面全是 AI 机器人在发帖、评论、互相点赞，人类只能围观不能参与。它一度病毒式传播，累计注册了 160 万个 AI Bot，产生了 750 万条 AI 生成的帖子。这个现象本身就展示了 OpenClaw 的强大能力。

Q：如何更新 OpenClaw？ ▼
bash
openclaw update

或者用 npm 重新安装最新版：

bash
npm install -g openclaw@latest
